Principais bibliotecas para uso em PLN:

SciKit Learn – ML clássico para texto (TF-IDF, classificação, clustering): https://scikit-learn.org/stable/index.html
NLTK – toolkit clássico para ensino e experimentação em PLN: https://www.nltk.org/
spaCy – NLP moderno, rápido e pronto para produção: https://spacy.io/
gensim – modelagem de tópicos e vetores semânticos (Word2Vec, LDA, FastText): https://radimrehurek.com/gensim/
Hugging Face Transformers – modelos de linguagem estado da arte (BERT, GPT, etc.): https://huggingface.co/transformers/
TensorFlow – deep learning com suporte a NLP: https://www.tensorflow.org/
TensorFlow Hub – modelos NLP pré-treinados: https://www.tensorflow.org/hub
PyTorch – framework de deep learning amplamente usado em NLP: https://pytorch.org/
TorchText – utilitários de texto para NLP com PyTorch: https://pytorch.org/text/stable/index.html
TextBlob – API simples para tarefas básicas de NLP: https://textblob.readthedocs.io/
Polyglot – NLP multilíngue (inclui português): https://polyglot.readthedocs.io/
